# TimeSwap

TimeSwap is a Credit-Based Skill Exchange Platform built using Flask, SQLite, HTML, CSS, and Bootstrap.

## Features

* User Registration & Login
* Credit-Based Marketplace
* Task Posting with Rewards
* Answer Submission
* Reputation System
* File Attachments
* Profile Management
* Community Learning Platform

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* Bootstrap

## Future Enhancements

* AI Skill Matching
* Real-Time Chat
* Leaderboard
* Cloud File Storage
